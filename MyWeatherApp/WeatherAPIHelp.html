Welcome My Weather Application help

This application contains two tabs home and another is the settings.

Home Tab:

In the home tab user can add his bookmark location by clicking the "+" on top right hand. On clicking the user is navigated to city screen.

Here the user can either search in the search bar the name of the city, or can "long press" on a particular location. The pin automatically comes when ever the user long press on a location.

User can add to bookmark location by clicking the annotation, and press the "+" add button to mark the location as favourite.

After adding the favourite location user is navigated to the main home screen, where the location is stored persistently and shown in the list.

On clicking the each location user is navigated to city screen, here the user can see current and next 5 days forecast, temperature, humidity, rain of chance, wind speed.

Settings Tabs:

1.setting tab has help.

2. Change unit/metric for temperature either Celsius or Fahrenheit

3. Reset all bookmarked Location, which will reset all previously bookmarked location.
